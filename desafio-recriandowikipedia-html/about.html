DIO

A DIO (Digital Innovation One) é um programa de bolsas de estudo gratuitas promovidas por grandes empresas para você aprender e praticar novas tecnologias, além de te conectar com a sua próxima oportunidade profissional.

Formação HTML Developer

Para começar uma carreira como Front-end a Formação HTML Developer da DIO é perfeita para começar essa jornada. Através dela é possível entender como trabalhar com HTML, a base para criar as primeiras páginas WEB! Aprender na prática a como estruturar a marcação de páginas e como funciona cada tag, conhecer os elementos semânticos e outras boas práticas na hora de construir a primeira página.

(voltar)

Desafio Proposto

O professor Diogo Mainardes propôs aos seus alunos da Formação HTML Developer - DIO que se recriasse uma página da Wikipédia para treinar-nos com relação a estrutura do site e também aplicar os conhecimentos adquiridos sobre semântica e acessibilidade, utilizando as tags e outros recursos HTML aprendidos no curso.
